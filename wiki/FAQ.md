# Frequently Asked Questions

## What is FerroTeX?

FerroTeX is a high-performance LaTeX language server and tooling platform written in Rust. It provides IDE features like code completion, diagnostics, and integrated PDF preview.

## What editors are supported?

Currently, FerroTeX provides a VS Code extension. The language server can work with any LSP-compatible editor.

## Does FerroTeX replace my TeX distribution?

No. FerroTeX works alongside TeX Live, MiKTeX, or Tectonic. It provides IDE features but relies on your TeX distribution for compilation.

## How do I report a bug?

Please open an issue on [GitHub Issues](https://github.com/jxoesneon/FerroTeX/issues).

## Is FerroTeX open source?

Yes! FerroTeX is dual-licensed under Apache-2.0 and MIT.
